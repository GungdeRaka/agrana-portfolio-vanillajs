# 🚀 AGRANA Web Agency Bali | High-Conversion Portfolio

A professional, high-performance web portfolio built to promote digital services for SMEs (UKM) in Bali. This project demonstrates a blend of modern frontend engineering, aesthetic UI/UX design, and a conversion-oriented business mindset.

**Live Demo:** [https://agrana-digital.web.app/]

---

## 💎 Key Highlights for Recruiters

- **Performance First:** Optimized with **Vite** and minimal dependencies for near-instant load times.
- **Modern Tech Stack:** Leveraging **Tailwind CSS (v3.4/v4)** for a utility-first, highly maintainable design system.
- **Custom Interactivity:** Handcrafted **Vanilla JavaScript** logic for a smooth mobile experience and dynamic typewriter effects, avoiding heavy third-party libraries.
- **Aesthetic Branding:** Custom-defined color palette (`darkbg`, `accent1`, `accent2`) and synchronized animations (floating, pulsing) that reflect a premium agency feel.
- **Responsive & Accessible:** Fully mobile-first design with smooth scroll behavior and accessible typography (Inter).

## 🌟 Core Features

### 1. Dynamic Hero Section
Features a smart **Typewriter Effect** that cycles through various business niches (Travel, Restaurant, Property, etc.), showcasing the versatility of the agency.
### 2. Custom Mobile Navigation
A bespoke mobile menu implementation with:
- Synchronized CSS transitions (Slide + Fade).
- Rotating SVG icons (Hamburger to Close transition).
- Automatic menu collapse on link selection.
### 3. Service Showcase
Strategically categorized services (WordPress vs. Custom Laravel Systems) to demonstrate technical breadth—from CMS management to complex database architecture.
### 4. Interactive Portfolio Gallery
A curated selection of real-world mockups (Inventory Systems, Cafe Profiles, Travel Landing Pages) with hover-state scaling and glassmorphism containers.

## 🚀 Getting Started

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/GungdeRaka/my-portfolio-agrana.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run development server:**
   ```bash
   npm run dev
   ```
4. **Build for production:**
   ```bash
   npm run build
   ```
